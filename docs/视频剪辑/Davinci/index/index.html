入门#

0.创建一个新项目

1.设置视频质量 file > Project setting > Master setting

  • timeline format 视频源质量
  • video Monitor format 预览质量

2.添加素材 Matser: 素材区域

  • 从文件中选择
  • 将素材拖入素材区域

3.创建时间线

  • 将素材拖入时间线区域

素材剪切

  • 点击开始时间+i 素材开始部分（也可鼠标拖动选择）
  • 点击开始时间+o 素材结束部分（也可鼠标拖动选择）
  • f9将选取的素材放到时间线
