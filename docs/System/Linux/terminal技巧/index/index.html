文件与路径操作#

路径

pwd         输出当前路径
cd          切换到：
.                 当前路径/刷新文件
..                上一级目录
/                 根目录
~                 当前用户home
*                 *慎用* xx目录下的所有文件 /xx/*
ls -al      列出当前路径下的所有文件/文件夹
ll
mkdir       创建路径
-m          配置权限
-p          自行创建多级目录
rm [name]   删除文件/路径
-r          递归删除(不为空的文件夹)
\rm         root用户时-r不询问
cp [fromdir] [todir]     复制文件/路径
-r                       复制路径下所有文件

文件

touch [filename] 创建文件
find             查找文件

权限#

chown [-cfhvR] [--help] [--version] user[:group] file...
chown -R leoy:leoy  [dir] | [filename] 授权文件或路径给用户组leoy的leoy

Terminal快捷键#

ctrl+左右方向键 左(右)移动一个单词
shift+上下方向键 上下滚动终端
shift`+`pageup/pagedown` 上下滚动终端(一页)
home` 移动到行首
end` 移动到行尾
ctrl+? 撤销
ctrl+w 删除到字首
alt+d 删除到字首
ctrl+u 删除到行首
ctrl+k 删除到行尾
ctrl+y 粘贴删除操作的字符

压缩/解压/打包#

『.tar, .tar.gz, .tgz, .gz, .Z, .bz2』

compress        old
gzip/zcat       popular
bzip2/bzcat     更好压缩比
tar             神器
-v              显示解压过程
-C              指定解压目录
-f              待处理文件
-c              创建打包(压缩)
-t              预览压缩文件内容
-x              解压
                解压*.tar.bz2  -j
                    *.tar.xz   -J
tar -jcv -f filename.tar.bz2            压缩
tar -jtv -f filename.tar.bz2            查询
tar -jxv -f filename.tar.bz2 -C [todir] 解压缩
                解压*.tar.gz -z
tar -zcv -f filename.tar.gz            压缩
tar -ztv -f filename.tar.gz            查询
tar -zxv -f filename.tar.gz -C [todir] 解压缩 -C未指定则不改变名字

curl#

curl [网址]   访问网页（返回html源码）
curl -o [文件名] https://juejin.im     # -o：将文件保存为命令行中指定的文件名的文件中
curl -O URL1 -O URL2                # -O：使用URL中默认的文件名保存文件到本地。 同一站点下多个文件，curl会尝试重用链接(connection)

dig | nslookup#

# 查看 host最快的ip
dig github.com
nslookup github.com
# 将查出的ip写到hosts中

系统服务#

systemctl enable xxx        开机启动xxx服务
systemctl status xxx        启动xxx服务
systemctl start xxx         查看xxx服务运行状态
service xxx restart         重启xxx服务
journalctl --unit elasticsearch
-r 时间倒序（先显示最近的）

系统信息#

uname -a            Linux系统版本
lsb_release -a
free            内存信息 默认单位kb
-m              mb为单位
df -h           磁盘空间情况

网络信息#

telnet [ip | host] [port]   测试服务某个端口是否通畅

安全防护#

w           查看当前登陆的所有用户
who am i    查询自己的 pts代号
pkill -kill -t pts/0  踢掉 pts/0的用户
last        查询历史登陆用户
whois       查询ip信息（不一定准确，国内的假公网IP等原因）
history     查询命令历史
lsof        网络进程信息
-i :port     查询某个端口的网络进程
netstat -p  网络进程信息
netstat -anp|grep 80  
shutdown -h now  关闭机器
systemctl poweroff 关闭机器

文件监听数量达到上限#

# 查看当前文件监听数量
cat /proc/sys/fs/inotify/max_user_watches
# 默认应该是8162
# 修改上限
sudo vim /etc/sysctl.conf
# 文件最后添加
fs.inotify.max_user_watches=524288
# 刷新配置
sudo sysctl -p
