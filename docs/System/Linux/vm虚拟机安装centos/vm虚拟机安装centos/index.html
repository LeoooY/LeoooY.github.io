vm虚拟机安装centos

准备#

  • vmware workstation Pro 14
  • Linux 镜像文件
    • CentOS | 官网

开始#

安装vmware centOS 镜像: centos 7 minimal 1780

创建虚拟机#

1 分配机器配置 选择稍后创建iso

2 挂载DVD驱动，选择centos镜像

3 启动-安装centos

安装centos#

系统选项

  • 选择语言
  • 选择分区

用户选项

  • 设置root 用户密码
  • 设置普通用户/管理员 用户密码

等待安装完成
