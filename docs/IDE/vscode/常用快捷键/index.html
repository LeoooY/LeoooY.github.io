常用快捷键

通用#

f2 重命名变量/文件/文件夹 f5 允许代码/任务

光标位置#

上/下/左/右 移动光标 ctrl + 左/右 移动到下一个英文单词/空格/符号(连续的中文被认为是一个单词) home/end 移动到行首/行尾

ctrl + home/end 移动到文件首/文件尾 PgUp/PgDn 移动到页首/页尾/翻页

编辑器#

ctrl + n 打开新窗口 ctrl + shift + n 打开新编辑器

ctrl + w 关闭窗口 ctrl + k + ctrl + w 关闭全部窗口

ctrl + s 保存(设置fucos变化即保存则不需要频繁保存)

ctrl + shift + p 执行vscode任务/插件功能

ctrl + ` 打开/关闭vs终端 ctrl + b 打开/关闭vs侧边栏(可能与md插件加粗冲突)

文件/文件夹#

ctrl + 0 打开文件 ctrl + k + ctrl + 0 打开文件夹 ctrl + p 搜索文件

alt + 鼠标左键 在右侧打开文件

鼠标左键双击文件 切换常驻打开 鼠标左键单击文件 临时打开(会被下一个临时打开的替换，有改动时候变为常驻打开)

代码操作#

ctrl + / 单行注释 shift + alt + / 多行注释

ctrl + f 搜索当前 ctrl + g 跳转到x行

ctrl + c / ctrl + Insrt 复制 ctrl + v / shift + Insrt 粘贴

ctrl + z 撤销 ctrl + shift + z 取消撤销

alt + 上/下 上下移动当前行代码 alt + shift + 鼠标 多选行

ctrl + alt + l Eslint自动修复
