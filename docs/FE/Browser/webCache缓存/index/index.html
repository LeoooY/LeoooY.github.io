浏览器storage#

localStorage和sessionStorage

  • localStorage
兼容性较差
网络快、协商缓存响应快的时候304比localStorage更快(移动端目前一般localStorage比304快还是因为网速相对较慢的原因)
不能缓存css
不清除的话会一直存储在浏览器中
理解为本地存储的公共资源，以域名进行划分，等价不同名的域名localStorage不互通，可能会引起存储资源的浪费
  • sessionStorage
存储在特定的会话中，关闭浏览器会清除数据
