- Don't return plain text
- Avoid using verbs in URIs   // url中避免使用动词
- Use plural resource nouns  // 使用名词复数
- Return error details in the response body // response中包含错误信息
eg： 
{
    "error": "Invalid payoad.",
    "detail": {
        "surname": "This field is required."
    }
}
- Pay attention to status codes   // 注意status code，返回和状态对应的有意义的code
- Don't nest resources  
bad : GET: /authors/12/articles/ 
good: GET: /articles/?author_id=12
- Handle trailing slashes gracefully  //少一个斜杠可能会导致500 server error
bad: POST: /entities
good: POST: /entities/
