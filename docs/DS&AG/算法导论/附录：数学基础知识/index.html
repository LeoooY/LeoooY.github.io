附录A： 求和（计算和式与求其界）#

当一个算法包含循环控制结构时，例如while和for循环，我们可以将算法的运行时间表示为每一次执行循环体所花时间之和。

A.1 求和公式及其性质#

发散、收敛、绝对收敛级数 线性性质 等差级数 平方和立方和 几何级数 调和级数 级数积分与微分 裂项级数 乘积

A.2 确定求和的界#
